# predictive-maintenance-ml-dashboard
# 🔧 Predictive Maintenance using Machine Learning + Power BI Dashboard

## 📌 Project Overview
This project focuses on predicting machine failures using machine learning and visualizing insights through an interactive Power BI dashboard.

The goal is to help maintenance teams identify high-risk machines and take proactive actions.

---

## 🚀 Tools & Technologies Used

- Python (Pandas, NumPy, Scikit-learn)
- Machine Learning (Random Forest, Model Evaluation)
- SQL (Data extraction & analysis)
- Power BI (Dashboard & Visualization)
- Google Colab (Development environment)

---

## 📊 Business Problem

Machine failures in industrial environments lead to costly downtime.

✅ This project predicts failures  
✅ Identifies high-risk machines  
✅ Enables proactive maintenance decisions  

---

## ⚙️ Workflow

### 1️⃣ Data Collection & Cleaning
- Handled missing values
- Converted data types
- Removed outliers

### 2️⃣ SQL Analysis
- Queried dataset using SQL
- Derived insights (failure count, distribution)

### 3️⃣ Feature Engineering
- Selected key features:
  - pressure
  - vibration
  - noise_level
  - rpm

### 4️⃣ Model Building
- Used Random Forest Classifier
- Handled class imbalance
- Evaluated using Accuracy & Confusion Matrix

### 5️⃣ Prediction Output
- Generated:
  - Predicted Failure
  - Failure Probability
  - Risk Band (Low / Medium / High)

---

## 📊 Power BI Dashboard

### ✅ Key Features:
- KPI Metrics (Total records, failure rate, high-risk machines)
- Risk Band Distribution
- Failure Pattern Analysis (Pressure vs Vibration)
- Interactive filters
- Top Risk Machines table (actionable insights)

---

## 📈 Key Insights

- High pressure and vibration increase failure probability
- Majority machines fall under Medium risk
- High-risk machines require immediate attention
